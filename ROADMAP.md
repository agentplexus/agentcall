# AgentCall Roadmap

This roadmap documents planned and completed features for this project.

The format is based on and generated by [Structured Roadmap](https://github.com/grokify/structured-roadmap), which provides a machine-readable JSON intermediate representation with deterministic Markdown generation.

## Overview

AgentCall is a Claude Code MCP plugin that enables voice calls via phone, built with the agentplexus stack (omnivoice, omnivoice-twilio, mcpruntime). Start a task, walk away. Your phone rings when Claude is done, stuck, or needs a decision.

## Technology Stack

- **omnivoice** - Voice abstraction layer (TTS, STT, Transport, CallSystem interfaces)
- **omnivoice-twilio** - Twilio implementation of omnivoice interfaces
- **mcpruntime** - MCP server runtime with built-in ngrok integration
- **assistantkit** - Multi-tool plugin configuration management (Claude, Kiro, Gemini)

## Multi-Tool Support

AgentCall supports multiple AI coding assistants through the assistantkit bundle API:

- **Claude Code** - Plugin manifest, skills, commands, hooks
- **AWS Kiro CLI** - Agent definitions, MCP server configuration
- **Gemini CLI** - Extension manifest, TOML agents
